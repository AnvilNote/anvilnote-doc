# 機能

![AnvilNote エディタ](assets/screenshots/ja.png)

## 執筆

- 見出し、リスト、表、画像に対応したブロック単位の編集
- 数式
- シンタックスハイライト付きのコードブロック
- 長いドキュメントを移動しやすくするドキュメントアウトライン

## テンプレート

白紙から始める代わりに、レポート・講義ノート・学術記事のテンプレートから始められます。テンプレートはエクスポートと同じ Typst のレンダリングパイプラインを使用しているため、プレビューは最終的な PDF と一致します。

## エクスポート

- **PDF エクスポート**：[Typst](https://typst.app/) をベースに、高速かつ高品質な出力
- **DOCX エクスポート**：Word 文書を必要とする相手との共有に

## オフラインファースト

- ローカルデスクトップ利用にログイン不要
- ローカルデスクトップ利用は外部クラウドサービスに依存しません
- デスクトップアプリは必要なツールをバンドル済みで、Node.js や Typst の別途インストールは不要です
