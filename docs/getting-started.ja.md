# はじめに

## 1. デスクトップアプリをダウンロード

[anvilnote-desktop のリリースページ](https://github.com/AnvilNote/anvilnote-desktop/releases) から最新版を入手してください。macOS（`.dmg`、`.pkg`）と Linux（`.deb`、`.AppImage`）向けのインストーラーを用意しています。

アカウント登録やログインは不要です。インストールしてすぐに使い始められます。

!!! note
    現在の macOS ビルドはまだコード署名・公証されていないため、初回起動時に macOS のセキュリティ警告が表示されることがあります。これは開発初期段階では想定内の動作です。未署名ビルドの開き方は [anvilnote-desktop の README](https://github.com/AnvilNote/anvilnote-desktop) を参照してください。

## 2. ワークスペースを開く

AnvilNote を起動します。アプリはすべてあなたのマシン上で動作し、執筆・編集・エクスポートにネットワーク接続は不要です。

## 3. 最初のドキュメントを作成

- サイドバーの「ドキュメント」をクリックして新規ドキュメントを作成します。
- あるいは「テンプレート」パネルから、レポート・講義ノート・記事などのテンプレートを選んで始めることもできます。

## 4. 執筆する

エディタは見出し、リスト、表、画像、コードブロック、数式を通常の文章とあわせてサポートしており、長文の構造化ドキュメントに必要な機能が揃っています。

## 5. PDF にエクスポート

準備ができたら、ツールバーの「PDF をエクスポート」を使います。エクスポートは [Typst](https://typst.app/) によって処理されるため高速で、選んだテンプレートに沿ったレイアウトが保たれます。
