# 開始使用

## 1. 下載桌面版

從 [anvilnote-desktop 發佈頁面](https://github.com/AnvilNote/anvilnote-desktop/releases) 取得最新版本。目前提供 macOS（`.dmg`、`.pkg`）與 Linux（`.deb`、`.AppImage`）安裝檔。

不需要帳號或登入，安裝完成即可開啟使用。

!!! note
    目前 macOS 版本尚未經過程式碼簽署或公證，因此首次啟動時 macOS 可能會顯示安全性警告，這在早期開發階段是預期狀況。開啟未簽署版本的方法請參閱 [anvilnote-desktop 的 README](https://github.com/AnvilNote/anvilnote-desktop)。

## 2. 開啟工作區

啟動 AnvilNote。應用程式完全在你的電腦端執行，寫作、編輯與輸出都不需要網路連線。

## 3. 建立第一份文件

- 在側邊欄點選「文件」，建立新文件。
- 或是從「範本」面板挑選一個範本作為起點（報告、講義、文章皆有）。

## 4. 開始寫作

編輯器支援標題、清單、表格、圖片、程式碼區塊與數學公式，搭配一般文字內容，滿足長篇結構化文件所需的一切。

## 5. 輸出成 PDF

完成後，使用工具列的「匯出 PDF」。輸出流程以 [Typst](https://typst.app/) 為核心，速度快，且版面會與你選擇的範本保持一致。
