# Начало работы

## 1. Скачайте десктопное приложение

Загрузите последнюю версию со [страницы релизов anvilnote-desktop](https://github.com/AnvilNote/anvilnote-desktop/releases). Доступны установщики для macOS (`.dmg`, `.pkg`) и Linux (`.deb`, `.AppImage`).

Учётная запись или вход в систему не требуются — установите и открывайте.

!!! note
    Текущие сборки для macOS пока не подписаны и не нотаризованы, поэтому при первом запуске macOS может показать предупреждение безопасности. На ранней стадии разработки это ожидаемо. Как открыть неподписанную сборку — см. [README anvilnote-desktop](https://github.com/AnvilNote/anvilnote-desktop).

## 2. Откройте рабочее пространство

Запустите AnvilNote. Приложение полностью работает на вашем устройстве — для письма, редактирования и экспорта подключение к сети не требуется.

## 3. Создайте первый документ

- Нажмите «Документы» на боковой панели и создайте новый документ.
- Или начните с шаблона на панели «Шаблоны», если нужна готовая структура (отчёт, конспект лекции, статья).

## 4. Пишите

Редактор поддерживает заголовки, списки, таблицы, изображения, блоки кода и формулы наряду с обычным текстом — всё необходимое для развёрнутого структурированного документа.

## 5. Экспортируйте в PDF

Когда будете готовы, используйте «Экспорт в PDF» на панели инструментов. Экспорт работает на основе [Typst](https://typst.app/), поэтому он быстрый, а вёрстка остаётся согласованной с выбранным шаблоном.
