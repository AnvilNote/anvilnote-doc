# 시작하기

## 1. 데스크톱 앱 다운로드

[anvilnote-desktop 릴리스 페이지](https://github.com/AnvilNote/anvilnote-desktop/releases)에서 최신 버전을 받으세요. macOS(`.dmg`, `.pkg`)와 Linux(`.deb`, `.AppImage`) 설치 파일을 제공합니다.

계정이나 로그인이 필요 없습니다. 설치 후 바로 사용할 수 있습니다.

!!! note
    현재 macOS 빌드는 아직 코드 서명이나 공증을 거치지 않았기 때문에, 처음 실행할 때 macOS 보안 경고가 표시될 수 있습니다. 초기 개발 단계에서는 예상되는 동작입니다. 서명되지 않은 빌드를 여는 방법은 [anvilnote-desktop README](https://github.com/AnvilNote/anvilnote-desktop)를 참고하세요.

## 2. 워크스페이스 열기

AnvilNote를 실행하세요. 앱은 전적으로 사용자의 기기에서 동작하며, 작성·편집·내보내기 모두 네트워크 연결이 필요하지 않습니다.

## 3. 첫 문서 만들기

- 사이드바에서 "문서"를 클릭해 새 문서를 만듭니다.
- 또는 "템플릿" 패널에서 보고서, 강의 노트, 아티클 등 구조화된 템플릿을 선택해 시작할 수 있습니다.

## 4. 작성하기

에디터는 일반 텍스트와 함께 제목, 목록, 표, 이미지, 코드 블록, 수식을 지원하여 장문의 구조화된 문서 작성에 필요한 요소를 모두 갖추고 있습니다.

## 5. PDF로 내보내기

작성이 끝나면 툴바의 "PDF 내보내기"를 사용하세요. 내보내기는 [Typst](https://typst.app/) 기반으로 처리되어 빠르며, 선택한 템플릿과 일관된 레이아웃을 유지합니다.
