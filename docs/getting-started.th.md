# เริ่มต้นใช้งาน

## 1. ดาวน์โหลดแอปเดสก์ท็อป

ดาวน์โหลดเวอร์ชันล่าสุดได้ที่ [หน้ารีลีสของ anvilnote-desktop](https://github.com/AnvilNote/anvilnote-desktop/releases) มีตัวติดตั้งสำหรับ macOS (`.dmg`, `.pkg`) และ Linux (`.deb`, `.AppImage`)

ไม่ต้องสร้างบัญชีหรือเข้าสู่ระบบ ติดตั้งแล้วเปิดใช้งานได้ทันที

!!! note
    บิลด์ macOS ปัจจุบันยังไม่ได้ code sign หรือ notarize ดังนั้น macOS อาจแสดงคำเตือนด้านความปลอดภัยเมื่อเปิดใช้งานครั้งแรก ซึ่งเป็นเรื่องปกติในช่วงพัฒนาระยะแรก วิธีเปิดบิลด์ที่ยังไม่ได้เซ็นชื่อดูได้ที่ [README ของ anvilnote-desktop](https://github.com/AnvilNote/anvilnote-desktop)

## 2. เปิดพื้นที่ทำงาน

เปิด AnvilNote แอปทำงานบนเครื่องของคุณทั้งหมด ไม่ต้องเชื่อมต่ออินเทอร์เน็ตสำหรับการเขียน แก้ไข หรือส่งออกเอกสาร

## 3. สร้างเอกสารแรกของคุณ

- คลิก "เอกสาร" ในแถบด้านข้าง แล้วสร้างเอกสารใหม่
- หรือเริ่มจากเทมเพลตในแผง "เทมเพลต" หากต้องการจุดเริ่มต้นที่มีโครงสร้างอยู่แล้ว (รายงาน บันทึกการบรรยาย บทความ)

## 4. เขียนเอกสาร

เอดิเตอร์รองรับหัวข้อ รายการ ตาราง รูปภาพ บล็อกโค้ด และสูตรคณิตศาสตร์ ควบคู่กับข้อความทั่วไป ครบทุกอย่างที่ต้องใช้สำหรับเอกสารยาวที่มีโครงสร้าง

## 5. ส่งออกเป็น PDF

เมื่อพร้อมแล้ว ใช้ปุ่ม "ส่งออก PDF" บนแถบเครื่องมือ การส่งออกขับเคลื่อนด้วย [Typst](https://typst.app/) จึงรวดเร็ว และเลย์เอาต์จะสอดคล้องกับเทมเพลตที่คุณเลือกไว้
