# 功能特色

![AnvilNote 編輯器](assets/screenshots/zh-TW.png)

## 寫作

- 區塊式編輯，支援標題、清單、表格與圖片
- 數學公式
- 具語法高亮的程式碼區塊
- 文件大綱，方便在長文件中導覽

## 範本

從報告、講義或學術文章的範本開始，不必從空白頁寫起。範本透過與輸出相同的 Typst 排版流程渲染，預覽畫面會與最終 PDF 一致。

## 輸出

- **輸出成 PDF**，以 [Typst](https://typst.app/) 為核心，速度快、品質高
- **輸出成 DOCX**，方便分享給需要 Word 文件的對象

## 離線優先

- 本地桌面版使用無需登入
- 本地桌面版使用不依賴外部雲端服務
- 桌面版已內建所需工具，不需要另外安裝 Node.js 或 Typst

## 支援的介面語言

| 語言 | 語系代碼 |
| --- | --- |
| 英文 | `en` |
| 繁體中文 | `zh-TW` |
| 日文 | `ja` |
| 韓文 | `ko` |

| 俄文 | `ru` |

## 目前尚未支援的部分

AnvilNote 目前處於早期開發階段。已規劃與明確不規劃的項目（例如強制性雲端帳號），請參閱[路線圖](https://github.com/AnvilNote/anvilnote/blob/main/ROADMAP.md)。
