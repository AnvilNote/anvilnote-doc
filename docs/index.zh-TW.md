# AnvilNote

AnvilNote 是一款離線優先的寫作與筆記應用程式，專為長篇筆記、講義、報告與學術文件而設計。它提供類似 Notion 的寫作體驗，但更著重於結構化文件輸出、範本、字型、公式、程式碼區塊，以及輸出成 PDF。

![AnvilNote 編輯器](assets/screenshots/zh-TW.png)

## 為什麼選擇 AnvilNote

- **預設離線優先。** 筆記資料存在你自己的電腦端。
- **本地桌面版使用無需登入。**
- **為長篇寫作而生** —— 講義、報告、學術文件，不只是短筆記。
- **公式、程式碼區塊、範本與輸出成 PDF** 是核心功能。
- **以 Typst 為核心的排版引擎**，提供快速且高品質的 PDF 輸出。
- **桌面版已內建所需工具。** 不需要另外安裝 Node.js 或 Typst。
