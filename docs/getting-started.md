# Getting Started

## 1. Download the desktop app

Grab the latest build from the [anvilnote-desktop releases page](https://github.com/AnvilNote/anvilnote-desktop/releases). Installers are available for macOS (`.dmg`, `.pkg`) and Linux (`.deb`, `.AppImage`).

No account or login is required — install it and open it.

!!! note
    Current macOS builds are not yet code signed or notarized, so macOS may show a security warning on first launch. This is expected during early development. See the [anvilnote-desktop README](https://github.com/AnvilNote/anvilnote-desktop) for how to open an unsigned build.

## 2. Open the workspace

Launch AnvilNote. The app runs entirely on your machine — no network connection is required to write, edit, or export.

## 3. Create your first document

- Click **Documents** in the sidebar, then create a new document.
- Or start from a template in the **Templates** panel if you want a structured starting point (report, lecture notes, article).

## 4. Write

The editor supports headings, lists, tables, images, code blocks, and math formulas alongside regular text — everything needed for a long-form, structured document.

## 5. Export to PDF

When you're ready, use **Export PDF** in the toolbar. Export is powered by [Typst](https://typst.app/), so output is fast and layout stays consistent with the template you picked.
