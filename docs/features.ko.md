# 기능

![AnvilNote 에디터](assets/screenshots/ko.png)

## 작성

- 제목, 목록, 표, 이미지를 지원하는 블록 기반 편집
- 수식
- 구문 강조가 적용된 코드 블록
- 긴 문서 탐색을 위한 문서 개요

## 템플릿

빈 페이지 대신 보고서, 강의 노트, 학술 아티클용 템플릿에서 시작할 수 있습니다. 템플릿은 내보내기와 동일한 Typst 렌더링 파이프라인을 사용하므로, 미리보기가 최종 PDF와 일치합니다.

## 내보내기

- **PDF 내보내기** — [Typst](https://typst.app/) 기반으로 빠르고 고품질의 출력을 제공합니다
- **DOCX 내보내기** — Word 문서가 필요한 상대와 공유할 때 사용합니다

## 오프라인 우선

- 로컬 데스크톱 사용 시 로그인 불필요
- 로컬 데스크톱 사용은 외부 클라우드 서비스에 의존하지 않습니다
- 데스크톱 앱에는 필요한 도구가 번들로 포함되어 있어 Node.js나 Typst를 별도로 설치할 필요가 없습니다

AnvilNote는 초기 개발 단계에 있습니다. 계획된 항목과 의도적으로 계획하지 않은 항목(예: 필수 클라우드 계정)은 [로드맵](https://github.com/AnvilNote/anvilnote/blob/main/ROADMAP.md)을 참고하세요.
